Catégorie Valeur CSS Description
Général auto Le navigateur détermine le curseur à afficher selon le contexte courant.
Ex. : vaut text lors du survol d'un texte.
default Curseur par défaut, habituellement une flèche.
none Aucun curseur affiché.
Liens & statut context-menu Un menu contextuel est disponible sous le curseur.
Seulement disponible sur IE 10 et supérieur sur Windows : bug 258960.
help Indique qu'une aide est disponible.
pointer Ex. : est utilisé lors du survol des liens, habituellement une main.
progress Le programme est occupé en fond mais l'utilisateur peut toujours intéragir avec l'interface (contrairement à wait).
wait Le programme est occupé (parfois un sablier ou une montre).
Sélection cell Indique que des cellules peuvent être sélectionnées.
crosshair Curseur en forme de croix, indique souvent une sélection de pixel sur une image.
text Indique que du texte peut être sélectionné.
vertical-text Indique que du texte vertical peut être sélectionné.
Drag and drop alias Indique qu'un alias ou un raccourci peut être créé.
copy Indique que quelque chose peut être copié.
move L'objet survolé peut être déplacé.
no-drop Curseur indiquant qu'il n'est pas possible de déposer quelque chose à cet endroit.
bug 275173 sur Windows, "no-drop est identique à not-allowed".
not-allowed Curseur indiquant que quelque chose ne peut pas être fait.
Redimensionnement & défilement all-scroll Cursor indiquant qu'il est possible de faire défiler dans n'importe quelle direction.
bug 275174 sur Windows, "all-scroll est identique à move".
col-resize L'élément peut être redimensionné horizontalement. Souvent représenté par des flèches pointant à gauche et à droite séparées par une barre verticale.
row-resize L'élément peut être redimensionné verticalement. Souvent représenté par des flèches pointant en haut et en bas séparées par une barre horizontale.
n-resize Un côté peut être déplacé. Par exemple, le curseur se-resize est utilisé quand le mouvement commence dans le coin en bas à droite de la boîte.
e-resize
s-resize
w-resize
ne-resize
nw-resize
se-resize
sw-resize
ew-resize Indique un redimensionnement bidirectionnel.
ns-resize
nesw-resize
nwse-resize
Zoom zoom-in Indique que quelque chose peut être zoomé ou dézoomé.
zoom-out
Saisie grab Indique que quelque chose peut être saisi (pour être déplacé par cliquer-glisser).
grabbing
